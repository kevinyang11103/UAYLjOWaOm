# Vue3宠物领养后台管理系统 java222

## 项目概述

本项目是一个基于Vue3的宠物领养后台管理系统，采用纯前端技术实现，无数据库依赖。系统提供了丰富的功能模块，页面美观，业务逻辑清晰，适用于学习Vue3 Composition API和后台管理系统开发。

## 技术栈

- 前端框架：Vue3
- UI组件库：Element Plus
- 构建工具：Vite
- 状态管理：Pinia
- 路由管理：Vue Router
- HTTP请求库：Axios
- 数据可视化：ECharts
- 日期处理：Day.js
- 工具库：Lodash
- CSS预处理器：Sass

## 功能模块

系统包含以下11个主要页面：

1. 仪表盘首页
2. 宠物分类管理
3. 宠物信息管理
4. 领养申请管理
5. 用户管理
6. 角色管理
7. 权限管理
8. 系统设置
9. 登录页
10. 注册页
11. 错误页面

此外，还包含15个以上的自定义组件，如分页组件等。

## 系统角色

系统实现了基于角色的权限管理，包含以下角色：

- 超级管理员
- 普通管理员
- 用户

## 运行环境

- Node.js版本：建议使用最新版
- 浏览器：支持现代浏览器，如Chrome、Firefox等

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖：`npm install`
3. 运行开发环境：`npm run dev`
4. 构建生产环境：`npm run build`

## 核心亮点

- 纯前端实现，无数据库依赖
- 使用Vue3 Composition API，便于学习和理解
